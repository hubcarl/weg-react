weg-react
=========

## eslint 使用

首先，安装ESLint。

$ npm i -g eslint

然后，安装Airbnb语法规则。

$ npm i -g eslint-config-airbnb

最后，在项目的根目录下新建一个.eslintrc文件，配置ESLint。

{
  "extends": "eslint-config-airbnb"
}

使用ESLint检查这个文件。
$ eslint index.js


## 安装插件

```
npm install -g weg
```

## 从组件生态安装 react
